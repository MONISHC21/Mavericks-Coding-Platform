# 🚀 Mavericks Coding Platform

An AI-powered developer skill assessment, learning, and recruitment platform that automates coding evaluation, analyzes resumes, and provides personalized learning paths.

---

## 🧠 Overview

Mavericks Coding Platform is a next-generation AI-driven ecosystem designed to bridge the gap between **learning, skill assessment, and recruitment**.

It solves the problem of **manual, biased, and inefficient coding evaluations** by providing:

* Automated skill assessment
* Resume analysis
* Real-time coding evaluation
* Personalized learning recommendations
* Recruiter-based candidate matching

---

## 🎯 Features

### 👨‍💻 User Features

* 📄 Resume Upload & AI Analysis
* 🧠 Skill Extraction (Tech stack detection)
* 💻 Live Coding Assessment (with output & error detection)
* 📊 AI Skill Evaluation (Score, Strengths, Weaknesses)
* 📚 Personalized Learning Path

## 🏗️ Architecture

The platform uses a **multi-agent AI architecture**:

* **Profile Agent** → Builds user profile from resume
* **Resume Analysis Agent** → Extracts skills & scores resume
* **Assessment Agent** → Generates and evaluates coding questions
* **Code Execution Agent** → Runs code & detects errors
* **Recommendation Agent** → Suggests learning path
* **Tracker Agent** → Tracks progress
* **Engagement Agent** → Leaderboard & gamification

### 🔄 Workflow

Resume Upload → Skill Extraction → Coding Assessment → AI Analysis → Learning Path → Dashboard → Recruitment Matching
